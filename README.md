# 🎉 MegaEvent - Event Management System

A modern **Event Management Web Application** built using **React + TypeScript**, developed by **Jay Kamble**.  
This platform helps organizations and colleges manage events, user registrations, passes, and admin control efficiently — all in one place.

---

## 🚀 Overview
**MegaEvent** is a full-featured event management system designed to simplify and automate event handling.  
It allows users to register, receive event passes (with QR codes), and lets admins control everything from a single dashboard.

---

## 🧩 Features
- 🧑‍💻 **User Registration & Authentication** (Firebase Auth)
- 🎫 **QR Code Event Pass System**
- 🧾 **Refund / Notice Management**
- 📊 **Admin Dashboard for Event Monitoring**
- 📱 **Fully Responsive UI (Mobile + Desktop)**
- ⚡ **Built with React + TypeScript + Firebase**
